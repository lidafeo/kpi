1. Добавление нового значения

  • Перейдите на эту страницу с помощью навигационной панели, которая находится в верхней части страницы.

  • В списке показателей эффективности деятельности, который находится по левой стороне страницы, выберите тот, значение которого собираетесь добавить.

    Внимание! Если Вы хотите увидеть полное название показателя, наведите курсором на нужный показатель эффективности, должно отобразиться его полное название.

    Например, если Вы хотите добавить значение показателя Н.П.3, найдите и нажмите на "Н.П.3":

  • Затем справа от списка показателей эффективности (снизу, если Вы зашли на сайт с телефона или другого небольшого устройства) появится описание выбранного показателя, кнопка "+ Добавить значение", а также таблица с уже добавленными Вами ранее значениями (если таковые имеются). Для добавления нового значения нажмите на кнопку "+ Добавить значение".

    Внимание! Если вместо кнопки "+ Добавить значение" появилась надпись "Этот ПЭД недоступен для Вашей должности", то значение выбранного показателя не может быть добавлено для Вашей должности.

  • После нажатия на кнопку "+ Добавить значение" должна появиться форма для добавления нового значения. Ее необходимо заполнить данными:

    Тип - к какому типу относится событие/действие

    Дата реализации - дата события/действия, либо дата добавления значения при отсутствии события

    Значение - значение показателя эффективности

    Ссылка - ссылка на источник, подтверждающий получение значения показателя

    Пояснительная записка - текстовое сообщение, подтверждающее получение значения показателя

    Файл для подтверждения - файл, подтверждающий получение значения показателя

  • После ввода всех необходимых данных в форму, нажмите на кнопку "Сохранить".

  • Если значение было успешно добавлено, должна появиться таблица с добавленным значением.

2. Просмотр существующих значений

  • Перейдите на эту страницу с помощью навигационной панели, которая находится в верхней части страницы.

  • Страница содержит таблицу со всеми значениями, действительных на расчетный период, который также указан на странице.

    Внимание! Если значений в этот период нет, вместо таблицы выводитcя надпись "Нет отмеченных значений ПЭД".

    В таблице отображена следующая информация:

    Дата добавления - дата добавления значения в систему

    ПЭД - шифр показателя эффективности, значение которого было добавлено

    Дата реализации - дата события/действия, либо дата добавления значения при отсутствии события (указывается при добавлении)

    Значение - значение показателя эффективности

    Файл - ссылка на скачивание файла, подтверждающего получение значения

    Статус - "Выполнено" - если значение действительно, "Недействительно", если значение было отмечено руководителем подразделения как неверное

    Подробнее - ссылка на полную информацию о значении

  • Для получения полной информации о значении нажмите на ссылку "Подробнее" в таблице.

3. Просмотр баллов за расчетный период

  • Перейдите на эту страницу с помощью навигационной панели, которая находится в верхней части страницы.

  • Страница содержит таблицу с баллами за внесенные значения. Баллы расчитываются за расчетный период, который указан на странице.

    Внимание! Если значений в этот период нет, вместо таблицы выводитcя надпись "За выбранный период нет выполненных ПЭД".

    В таблице отображена следующая информация:

    Шифр - шифр показателя эффективности

    Показатель - полноназвание показателя эффективности

    Оценка - балл за показатель, расчитанный на основе добавленных значений
